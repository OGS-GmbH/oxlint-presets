# Getting started

## Installation

### Prerequisites

- Node.js version 18 or higher.
- A package manager: e.g. npm, pnpm, ...

::: code-group

```sh [npm]
$ npm add @ogs-gmbh/oxlint-presets
```

```sh [pnpm]
$ pnpm add @ogs-gmbh/oxlint-presets
```

```sh [yarn]
$ yarn add @ogs-gmbh/oxlint-presets
```

```sh [bun]
$ bun add @ogs-gmbh/oxlint-presets
```

:::
